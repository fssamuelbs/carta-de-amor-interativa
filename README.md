# 💌 Carta Interativa de Amor

Um template de carta romântica interativa feito com **HTML, CSS e JavaScript**.
Perfeito para surpreender alguém especial de forma criativa 💖

---

## ✨ Funcionalidades

* 💌 Carta personalizada em formato digital
* 💖 Animação de corações na tela
* 📸 Galeria de imagens
* 🎨 Design simples e elegante
* ⚡ 100% front-end (sem necessidade de backend)

---

## 🚀 Como usar

1. Clone o repositório:

```bash
git clone https://github.com/fssamuelbs/carta-interativa-de-amor.git
```

2. Abra o arquivo `index.html` no navegador

3. Personalize:

* ✏️ Textos da carta
* 📸 Imagens da galeria
* 💬 Nome da pessoa especial

---

## 🎨 Personalização

Substitua os seguintes elementos:

* `[Nome da pessoa]`
* `[Seu nome]`
* Imagens (`src` das tags `<img>`)
* Textos das páginas

---

## 🌐 Publicar como site (GitHub Pages)

1. Vá em **Settings** do repositório
2. Clique em **Pages**
3. Em **Branch**, selecione `main`
4. Clique em **Save**

Seu site ficará disponível em:


```
https://fssamuelbs.github.io/carta-interativa-de-amor
```
